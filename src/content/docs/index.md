# Welcome to Kent Schaeffer Documentation

This documentation site provides comprehensive guides, tutorials, and reference materials for various technical topics.

## Content Structure

Documentation follows the Diataxis framework with four main content types:

### Tutorials
Step-by-step learning-oriented guides for newcomers. Start here if you're new to a topic.

### How-To Guides
Task-oriented guides for solving specific problems. Use these when you know what you want to accomplish.

### Reference
Technical specifications and API documentation. Consult these for detailed information.

### Explanation
Conceptual understanding-oriented content. Read these to understand the "why" behind the technology.

## Getting Started

Browse the sidebar navigation to explore available documentation. Each section is organized by topic and content type.
