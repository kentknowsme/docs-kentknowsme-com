### Prerequisites

The tutorials in this collection assume you have a basic understanding of these topics (and have at least one of each tool installed):

- Basic MacOS (just about any Mac computer with a recent version of MacOS)
- The command-line (Terminal, zsh)
- Web Browser (Chrome, FireFox, Safari)
- Text Editor (VS Code, TextEdit, VIM, Nano, ...)
