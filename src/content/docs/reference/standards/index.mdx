## Documentation Philosophy

This site follows the [Diataxis framework](https://diataxis.fr/) for technical documentation:

- **Tutorials** teach through practical lessons
- **How-to guides** provide step-by-step instructions
- **Reference** material offers technical descriptions
- **Explanation** clarifies concepts and context
